# 🏫 How to Access School Structure Settings

## 📍 **Multiple Access Points**

You can access the School Structure Settings from **4 different places** in the UI:

---

## 1️⃣ **Direct URL Access**
```
http://localhost:3000/settings/school-structure
```
**Fastest way** - Bookmark this URL for quick access!

---

## 2️⃣ **Main Settings Page**
1. Go to: `http://localhost:3000/settings`
2. Look for the **🏫 School Structure** card
3. Click **"🔧 Manage Structure"** button

![Settings Page Card](https://via.placeholder.com/600x200/1f2937/10b981?text=🏫+School+Structure+Card+in+Settings)

---

## 3️⃣ **Navigation Sidebar** (Recommended)
1. Look at the left sidebar navigation
2. Scroll to **"Administration"** section
3. Click **"🏫 School Structure"**

**Features:**
- ✅ Active when you're on the page
- ✅ Always visible from any page
- ✅ Fastest navigation method

---

## 4️⃣ **Dashboard Quick Actions**
1. Go to Dashboard: `http://localhost:3000/dashboard`
2. Look for **Quick Actions** section
3. Find the **"🏫 School Structure"** card with "Admin" badge
4. Click on it

**Best for:** Administrators who frequently use the dashboard

---

## 🔧 **Quick Start Guide**

### **For First-Time Users**
1. **Access**: Go to `/settings/school-structure`
2. **Explore**: Click through Mediums, Classes, Sections tabs
3. **Try**: Add a test medium or section
4. **Export**: Click "📥 Export Config" to backup

### **For Daily Administration**
1. **Bookmark**: `/settings/school-structure` in your browser
2. **Sidebar**: Use the navigation sidebar for quick access
3. **Bulk**: Use checkboxes for multiple item operations
4. **Export**: Regularly export your configuration

## 🆘 **Troubleshooting**

### **Can't Access?**
1. Check your **user role**
2. Ensure you're **logged in**
3. Try **refreshing** the page
4. Clear **browser cache**

### **Page Not Loading?**
1. Check **internet connection**
2. Try **direct URL**: `/settings/school-structure`
3. **Restart** the application
4. Check **browser console** for errors

### **Changes Not Saving?**
1. **Export** your configuration first
2. **Refresh** and try again
3. Check if **required fields** are filled
4. Note: Changes are **in memory** only
