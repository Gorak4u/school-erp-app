Comprehensive icon library for the School Management ERP platform, providing **consistent**, **accessible**, and **scalable** icons that enhance **user experience** and **visual communication** across all applications.

## 🎨 **Icon System**

### **🔧 Icon Specifications**
```yaml
Design Principles:
  - Line weight: 2px
  - Corner radius: 2px
  - Grid: 24x24
  - Consistent style
  - Monochrome
  - Scalable

Technical Specifications:
  - Format: SVG
  - Size: 24x24 (default)
  - ViewBox: 0 0 24 24
  - Fill: None
  - Stroke: CurrentColor
  - Stroke-width: 2
  - Stroke-linecap: round
  - Stroke-linejoin: round

Accessibility:
  - ARIA labels
  - Semantic meaning
  - Screen reader support
  - High contrast support
  - RTL support
```

### **📏 Icon Sizes**
```yaml
Size Scale:
  Extra Small: 16px
  Small: 20px
  Medium: 24px (default)
  Large: 32px
  Extra Large: 48px
  Ultra Large: 64px

Usage Guidelines:
  - 16px: Inline text, tight spaces
  - 20px: Small buttons, compact UI
  - 24px: Default size, most common
  - 32px: Large buttons, headers
  - 48px: Hero icons, emphasis
  - 64px: Special cases, decorative

Responsive Behavior:
  - Scale proportionally
  - Maintain readability
  - Touch-friendly on mobile
  - Consistent visual weight
  - Optimize for each size
```
