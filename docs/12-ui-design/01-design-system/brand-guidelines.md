```yaml
  Font Family: 'Roboto', sans-serif
  - Rationale: Professional, reliable, accessible
  - Characteristics: Neutral, clear, functional

  Font Family: 'JetBrains Mono', monospace
  - Rationale: Technical, code, data
  - Characteristics: Clear, technical, precise

  Display: 96px - Hero titles, main branding
  Headline 1: 60px - Page titles, main headings
  Headline 2: 48px - Section headings
  Headline 3: 36px - Subsection headings
  Headline 4: 24px - Card titles
  Headline 5: 20px - Component headings
  Headline 6: 16px - Small headings
  Body 1: 16px - Primary body text
  Body 2: 14px - Secondary text
  Button: 14px - Button text
  Caption: 12px - Captions, footnotes
  Overline: 10px - Overlines, badges
```

```yaml
Writing Principles:
```
